# edm4hep-feedstock

[![Build AMD64](https://github.com/hep-forge/edm4hep-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/edm4hep-feedstock/actions/workflows/autoupload.amd64.yml)
[![Build ARM64](https://github.com/hep-forge/edm4hep-feedstock/actions/workflows/autoupload.arm64.yml/badge.svg)](https://github.com/hep-forge/edm4hep-feedstock/actions/workflows/autoupload.arm64.yml)

Feedstock for [edm4hep](https://github.com/key4hep/EDM4hep) — part of [hep-forge](https://anaconda.org/hep-forge).

A generic event data model for future HEP collider experiments.

## Install

```bash
conda install -c hep-forge -c conda-forge edm4hep
```
